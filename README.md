#프로젝트 이름: 
To‑Do List 웹 앱

#프로젝트 소개
이 프로젝트는 HTML, CSS, JavaScript를 이용해 만든 웹 To‑Do List이다.
숲을 테마로 디자인하였고 할 일을 입력하고 관리할 수 있는 기능을 구현했다.

  - 구현한 기능
	1. 숲 테마 적용
	전체 배경과 색상을 초록 계열로 구성
	리스트 항목 앞에 트리 아이콘을 두어 테마 설정
	
	2. 할 일 추가 기능
	입력창에 텍스트를 작성하고 “추가” 버튼을 누르면 목록에 표시됨
	총 할 일 개수도 상단에서 자동으로 업데이트됨
	
	3. 완료 체크 기능
	체크박스를 누르면 할 일이 “완료” 상태가 됨
	완료 시 항목이 회색 배경 + 취소선으로 변경됨
	
	4. 삭제 기능
	삭제 버튼을 누르면 해당 할 일이 목록에서 완전히 사라짐
	localStorage에서도 같이 제거됨
	
	5. 새로고침해도 데이터 유지
	localStorage를 사용하여 브라우저를 껐다 켜도 목록이 유지됨

  - 실행 방법
	프로젝트 폴더 안의 index.html파일을 크롬이나 마이크로엣지 같은 웹브라우저로 열고
	할 일을 작성하고 추가 버튼을 클릭한다.
	할 일이 완료되면 체크박스를 클릭하여 완료 처리한다.
	더 이상 해당 할 일을 보고 싶지 않으면 삭제를 클릭하여 제거한다.

  - 사용한 기술스택
	HTML: 웹페이지 구조 구성
	CSS: 숲 테마 UI 
	JavaScript: 할 일 추가, 완료 처리, 삭제 기능, localStorage 저장+불러오기

  - 기타 참고사항 (알림 권한 필요, 브라우저 제한 등)
	Chrome, Edge, Safari 등 대부분 브라우저에서 동작
